# 💕 LoveProgram — 节日祝福/表白弹窗小程序

一个用 Python + Tkinter 编写的浪漫表白小工具，通过满屏弹窗 + 爱心动画 + 口令解锁的方式，给 TA 一个甜蜜的惊喜。

## 效果预览

1. **爱心弹窗** — 屏幕上按心形轮廓依次弹出彩色关心弹窗
2. **随机炸屏** — 150 个弹窗随机铺满全屏，满屏都是爱
3. **口令解锁** — 弹出密码框，输入正确口令才能解锁
4. **漩涡吸收** — 所有弹窗以螺旋路径飞向屏幕中心并消失
5. **最终告白** — 显示告白文字，停留 5 秒后程序退出

## 运行方式

### 直接运行

```bash
python main.py
```

### 打包为 exe

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole main.py
```

打包后的可执行文件在 `dist/` 目录下，可以直接发给对方双击运行。

## 自定义

打开 `main.py` 顶部的参数区域即可自定义：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `HEART_STEP` | 爱心轮廓步长（度） | 3 |
| `HEART_SPEED` | 爱心弹窗间隔（ms） | 30 |
| `HEART_STAY` | 爱心停留时间（ms） | 2000 |
| `RANDOM_COUNT` | 随机弹窗数量 | 150 |
| `RANDOM_SPEED` | 随机弹窗间隔（ms） | 25 |
| `MESSAGES` | 弹窗显示的祝福短句 | 见代码 |
| `BG_COLORS` | 弹窗背景色列表 | 见代码 |

## 口令彩蛋

除了默认口令"本笨笨收到"，还支持：收到、收到啦、知道啦、爱你、我爱你 等变体。
